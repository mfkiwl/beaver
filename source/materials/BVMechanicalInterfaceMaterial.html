BVMechanicalInterfaceMaterial

buildconstruction:Undocumented Class

The BVMechanicalInterfaceMaterial has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Material for computing the displacement jump and the traction across a cohesive zone interface.
